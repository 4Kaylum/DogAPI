The Dog API

API Versions

Unsurprisingly, this API site has an API. Weird, I know, but it is what it is. As time goes on, I'm probably going to have to rewrite the API to be able to handle different kinds of requests and change the backend database and things like that. As such, the API has different versions. These will be listed below. I recommend the latest version, as that's always going to be the most supported version, but feel free to use any that you like. Click whatever version of API you want to use, and that'll take you to that version's documentation.
